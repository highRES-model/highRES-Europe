# highRES-Europe

The current working version of highRES-Europe including all GAMS code and necessary text files/GDX format input data for a 8760 hour model run. To run the model one needs GAMS/CPLEX installed and licensed, then download this repo, open highres.gpr and press play. Capacity units are in GW, energy in GWh and costs in millions of 2010 GBP.
